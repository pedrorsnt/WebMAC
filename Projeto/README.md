# ***iME*D** - Sistema de Gerenciamento de Farmácia

## 📌 Sobre o Projeto
O ***iME*D** é uma aplicação web desenvolvida com o intuito de otimizar o controle de recursos e estoques de uma farmácia. Com várias funcionalidades, o sistema possibilita o gerenciamento completo do ciclo de vida dos produtos, desde o cadastro de fabricantes até a aplicação de regras de venda e restrições de medicamentos.

## ⚙️ Funcionalidades Principais

* **Gestão de Medicamentos:** Cadastro, edição e remoção de remédios e demais produtos.
* **Controle de Estoque:** Monitoramento de quantidades disponíveis e alertas de estoque baixo.
* **Restrições de Compra:** Configuração de regras para venda, como exigência de retenção de receita médica, medicamentos de tarja preta e verificação de idade.
* **Precificação:** Definição de preços preços de venda e aplicação automática de desconto aos clientes.
* **Controle de Validade:** Registro dos prazos de validade de cada item e alerta de quando há medicação vencida em estoque.
